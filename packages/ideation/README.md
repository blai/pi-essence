# ideation

> Pi skills for building pi plugins.

## Install

```bash
pi install npm:ideation
```

## Skills

| Skill | Triggers when… |
|-------|----------------|
| `smith` | Creating, writing, scaffolding, or improving a pi **extension** (TypeScript module) |
| `coach` | Creating, writing, scaffolding, or improving a pi **skill** (SKILL.md) |

## Part of pi-essence

This package is one of three focused pi plugin bundles:

| Package | Contents |
|---------|----------|
| [`ideation`](../ideation) | `smith` + `coach` — build pi plugins |
| [`quality`](../quality) | `architect` + `poet` — prose & doc quality |
| [`presentation`](../presentation) | `gws` + `md-gdoc` — Google Workspace output |
