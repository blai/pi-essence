# quality

> Pi skills for prose and documentation quality.

## Install

```bash
pi install npm:quality
```

## Skills

| Skill | Triggers when… |
|-------|----------------|
| `architect` | Validating or improving AI-consumable markdown: SKILL.md, system prompts, commands |
| `poet` | Reviewing or improving human-facing prose: README, docs, changelogs, roadmaps |

## Part of pi-essence

This package is one of three focused pi plugin bundles:

| Package | Contents |
|---------|----------|
| [`ideation`](../ideation) | `smith` + `coach` — build pi plugins |
| [`quality`](../quality) | `architect` + `poet` — prose & doc quality |
| [`presentation`](../presentation) | `gws` + `md-gdoc` — Google Workspace output |
