# presentation

> Pi skills for Google Workspace output.

## Install

```bash
pi install npm:presentation
```

## Skills

| Skill | Triggers when… |
|-------|----------------|
| `gws` | Interacting with Google Workspace services via the `gws` CLI (Drive, Gmail, Sheets, Calendar, Docs, …) |
| `md-gdoc` | Converting a `.md` file to a Google Doc |

## Part of pi-essence

This package is one of three focused pi plugin bundles:

| Package | Contents |
|---------|----------|
| [`ideation`](../ideation) | `smith` + `coach` — build pi plugins |
| [`quality`](../quality) | `architect` + `poet` — prose & doc quality |
| [`presentation`](../presentation) | `gws` + `md-gdoc` — Google Workspace output |
